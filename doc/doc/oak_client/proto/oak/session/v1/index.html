    AttestationEndorsement contains statements that some entity (e.g., a hardware provider) vouches for the integrity of claims about the TEE or the software running on it. (see https://www.rfc-editor.org/rfc/rfc9334.html#name-endorsements).
  • AttestationEvidence
    AttestationEvidence contains all the information that originates in the TEE and can serve as claims about the Target Environment. The name is chosen to match the RATS terminology (see https://www.rfc-editor.org/rfc/rfc9334.html#name-evidence).
